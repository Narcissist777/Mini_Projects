# To-Do List Web Application

This is a simple web application for managing your tasks. You can add, delete, and mark tasks as completed. It's built using HTML, CSS, and JavaScript.

## Features

- Add tasks: Easily add new tasks to your to-do list.
- Delete tasks: Remove tasks that you no longer need.
- Mark tasks as completed: Keep track of your progress by marking tasks as done.
- Simple and intuitive interface: The user-friendly design makes it easy to manage your tasks.

## Usage

1. Clone the repository:

```
git clone https://github.com/Narcissist777/Mini_Projects.git
```

2. Open `index.html` in your web browser.

3. Start adding tasks to your to-do list!

- HTML
- CSS
